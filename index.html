jQuery.timezone v0.1.0

timezone is a tiny jQuery Plugin that fill a select element with current timezones and try to guess your timezone preselecting it. It has it own tests and can be installed with Bower.

Initializing

Initializing timezone is really simple as you only need to call the plugin in order to fill it with some default timezones. Quite convinient, huh?

  • Demo
  • JavaScript
  • HTML
$('#demo').timezone();
<select id="demo"></select>

Overwriting

If you don't like our cities you can make your owns!

  • Demo
  • JavaScript
  • HTML
var myCities = [
  {value: "-10.0", name: "Honolulu"},
  {value: "-7.0", name: "San Francisco"}
];

$('#demo-two-select').timezone(myCities);
<select id="demo-two-select"></select>
